mydotfiles
===

This is a collection of my dotfiles

Usage
===

There is a bootstrap shellscript that will install all your dependencies.

```
git clone git@github.com:rhoml/mydotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap.sh
```

Assumptions
===

* Emacs + Spacemacs
* Tmux + emacs mode
* ZSH
* iTerm2
